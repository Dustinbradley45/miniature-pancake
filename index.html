We Love Clip-Paths!

What is a Clip-Path?

  • Replaced the CSS property 'Clip'.

  • Defines the parameters (or path) of an element a developer wants to be seen/hidden.

  • Can use on images, divs, sections and other sematic elements.

  • Clip-paths can be used with transitions and animations

  • Not supported by IE and Edge

Hot Tip!

No matter how much of an element you clip it will still have the dimensions of the original coded element.

  • Circle Clip-Paths

  • Values needed: You have to pass in the radius value(in px or %). Optionally, you can define the location values as X, Y coordinates.

  • Syntax: clip-path:circle ([radius.value] *at [X.location Y.location]*)

  • Code as shown: clip-path: circle(100px).

  • *Property Optional*

  • Ellipse Clip-Paths

  • Values needed: shape length vlaue and then the height value(in px or %). Optionally, you can define the location values as X, Y coordinates.

  • Syntax: clip-path: ellipse ([length.value height.value] *at [X.location Y.location]*)

  • Code as shown: clip-path: ellipse (150px 70px at 50% 50%)

  • *Property Optional*

  • Polygon Clip-Paths

  • The Polygon property allows you to various X/Y coordinates to create a clip-path.

  • Code as shown: clip-path: polygon(0% 20%, 60% 20%, 60% 0%, 100% 50%, 60% 100%, 60% 80%, 0% 80%)

  • Code as shown: clip-path: polygon(40% 0%, 40% 20%, 100% 20%, 100% 80%, 40% 80%, 40% 100%, 0% 50%);

  • Defining Polygon Coordinates

  • See the Pen Clip-path: Polygon coordinates by Karen Menezes (@imohkay) on CodePen.

  • Note: There are really great clip-path polygon resources out create your own clip-path like Clippy!
