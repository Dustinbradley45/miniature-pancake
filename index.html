We Love Clip-Paths!

What is a Clip-Path?

A clip path is a way to 'cut' portions and sizes of elements on your webpage. In actuality, while it looks like it is cut, the portions are actually just hidden! In other words it defines what parts of an image is visible.

You can use clip-path properties on backgrounds, images, and most semantic elements out there (i.e. divs, sections, etc.)

Tips for CSS clip-paths:

  • Since clip-path does not actually cut any portion of the element the space that is then blank will still be registering to your browser as filled space! For instance, if you have a div element with a width and height of 500px, and use clip-path to change the shape, the newly created white space will still be counted as the total size of the originally shaped div you had created. You can see this by opening up your dev tools.

When Can I use Clip-Path

Clip-Path is compatible on most modern browsers, but is not compatible with IE or Edge.

  • Circle Clip-Paths

  • Values needed: You have to pass in the radius value(in px or %). Optionally, you can define the location value if needed.

  • Syntax: clip-path:circle ([radius.value] *at [x.location y.location]*)

  • Code as shown: clip-path: circle(100px).

  • *Property Optional*

  • Ellipse Clip-Paths

  • Using the ellipse property with clip-path will give you ELLIPSES! You can use this property by passing in the shape radius length and then the height (these values can be in either px or % value). Optionally, you can define the location value if needed!

  • Syntax: clip-path: ellipse ([length.value height.value] *at [X.location Y.location]*)

  • Code as shown: clip-path: ellipse (150px 70px at 50% 50%)

  • *Property Optional*

  • Polygon Clip-Paths

  • The Polygon property allows you to use polygons to create fun shapes of all varieties! For each 'point' of the polygon we set a percentage value for both the X and Y values. Note: There are really great clip-path polygon resources out there like Clippy!

  • Code as shown: clip-path: polygon(0% 20%, 60% 20%, 60% 0%, 100% 50%, 60% 100%, 60% 80%, 0% 80%)

  • Code as shown: clip-path: polygon(40% 0%, 40% 20%, 100% 20%, 100% 80%, 40% 80%, 40% 100%, 0% 50%);
